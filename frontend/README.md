# UI Recommender Frontend

## セットアップ
```bash
# 依存パッケージのインストール
npm install

# 開発サーバー起動
npm run dev
```
開発サーバーが http://localhost:5173 で起動します。

## 利用可能なコマンド
```bash
# 開発サーバー起動
npm run dev

# 本番ビルド
npm run build

# ビルド結果のプレビュー
npm run preview

# コードのリント
npm run lint
```

## 技術スタック
- React 19
- TypeScript
- Vite
